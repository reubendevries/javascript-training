# Data Types

## Different type of Data Types
1. Number -> floating point numbers, used for decimals and intergers

2. String -> Sequence of characters used for text

3. Logical type that can only be either true or false and is often used for taking decisions

4. Undefined -> Value taken by a variable that is not yet defined ('empty value')

5. Null -> Also means 'empty value'

6. Symbol (ES2015) -> Value that is unique and can't be changed

7. BigInt (ES2020) -> Larger integers than the Numbers type can hold.

We can use // to write comments, comments are important to either be used as a reminder to ourselves
or other developers on why we made the choices we did in our code.
